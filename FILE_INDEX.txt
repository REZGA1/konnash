DATABASE SCHEMA:
================

user_profile:
- id: INTEGER PRIMARY KEY
- store_name: TEXT
- phone: INTEGER
- country_code: TEXT
- activity_type: TEXT
- sector: TEXT
- initial_balance: REAL
- open_date: TEXT

transactions:
- id: INTEGER PRIMARY KEY AUTOINCREMENT
- type: TEXT (INCOME/EXPENSE)
- amount: REAL
- category: TEXT
- description: TEXT
- date: TEXT
- archive_id: INTEGER (0=current)

archives:
- id: INTEGER PRIMARY KEY AUTOINCREMENT
- total_balance: REAL
- total_income: REAL
- total_expense: REAL
- open_date: TEXT
- close_date: TEXT

QUICK START:
============

In MainActivity.java, you would use:

```java
// Initialize managers
ProfileManager profile = new ProfileManager(this);
FinanceManager finance = new FinanceManager(this);
ReportManager report = new ReportManager(this);

// First time - create user profile
profile.initializeProfile(
    "My Store",        // store name
    987654321,         // phone number (int)
    "+213",           // country code
    "Retail",         // activity type
    "Commerce",       // sector
    5000,            // initial balance
    "2024-04-02"     // open date
);

// Add transaction
finance.addIncome("1500", "Sales", "Product A", "2024-04-02");

// Get balance
Balance balance = finance.getCurrentBalance();

// Generate report
Report dailyReport = report.getTodayReport("2024-04-02");
```
